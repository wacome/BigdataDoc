# HBase

HBase 是一个分布式、可扩展、面向列的数据库，基于 Google 的 BigTable 构建，是 Hadoop 项目的子项目。HBase 不同于一般的关系数据库，它是一个适合于非结构化数据存储的数据库。另一个不同的是 HBase 基于 Hadoop 的 HDFS 来提供高可用性。
