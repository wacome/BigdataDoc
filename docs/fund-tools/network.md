## 1.购买代理服务：

付费的代理服务使用起来更稳定，速度更快，免费的也可以用，但是需要不够稳定。

在此推荐魔戒代理服务，非常便宜好用。

### 1.1 访问 [魔戒](https://www.mojie.cyou/) 

注册账号，登录。

<img src="../assets/mojie.png" alt="mojie" style="zoom:67%;" />

按流量付费就很棒！推荐购买130G流量的，足够使用。
